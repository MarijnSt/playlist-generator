## About project

This project is a simple implementation of the spotify API to create playlists for when I go for a run.
It is a laravel project that uses Inertia and Vue3 to create the frontend.

## How to install
- cp .env.example .env
- composer install
- php artisan key:generate
- php artisan migrate
- php artisan serve
- npm install
- npm run dev
